In order to install the MCI models library to your MATLAB installation perform following steps:

 * extract the "mci_models.zip" to your system at a permanent location
 * open MATLAB and add the library location to the MATLAB path via the "Set Path -> Add with Subfolder ... -> Open -> Save"
 * start Simulink and refresh the Library Browser by right-clicking in it or pressing F5
 * if problems with repositary informations are reported, just click on "Fix it"
